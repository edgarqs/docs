## Instalación

Para comenzar a usar RapidUI en tu proyecto, sigue estos pasos:

1. Clona el repositorio:

    ```bash
    git clone https://github.com/edgarqs/RapidUI.git
    cd RapidUI
    ```

2. Instala las dependencias:

    ```bash
    npm install
    ```

3. Compila los archivos SCSS a CSS:

    ```bash
    npm run build-css
    ```

## Uso

Para usar RapidUI en tu proyecto, simplemente incluye el siguiente enlace en tu archivo HTML:

```html
<link href="https://cdn.jsdelivr.net/gh/edgarqs/RapidUI/css/styles.css" rel="stylesheet">
```

### Ejemplo de Uso

Aquí tienes un ejemplo de cómo usar RapidUI en tu proyecto:

```html index.html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Project with RapidUI</title>
    <link href="https://cdn.jsdelivr.net/gh/edgarqs/RapidUI/css/styles.css" rel="stylesheet">
</head>
<body>
    <h1>Welcome to My Project with RapidUI</h1>
    <button class="btn btn-primary">Primary Button</button>
</body>
</html>
```

## Personalización

Puedes personalizar RapidUI modificando los archivos SCSS según tus necesidades. Los estilos y variables principales se encuentran en el archivo `variables.scss`.

## Contribuciones

Agradecemos cualquier contribución para mejorar RapidUI. Si tienes alguna sugerencia o quieres reportar un problema, sigue estos pasos:

1. Haz un fork del repositorio

2. Crea una rama para tu contribución:

    ```bash
    git checkout -b my-contribution
    ```

3. Realiza tus cambios y haz un commit:

    ```bash
    git commit -m "Description of my contribution"
    ```

4. Empuja tus cambios a la rama:

    ```bash
    git push origin my-contribution
    ```

5. Abre un Pull Request en GitHub.
