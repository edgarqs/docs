## Variables

The following variables are imported from the `_variables.scss` file:

- `$ru-font-family`: 'Poppins', sans-serif
- `$ru-font-size-base`: 16px
- `$ru-white`: #ffffff
- `$ru-black`: #000000
- `$ru-gray-light`: #f8f9fa
- `$ru-gray`: #6c757d
- `$ru-gray-dark`: #343a40
- `$ru-primary`: #007bff
- `$ru-secondary`: #6c757d
- `$ru-success`: #28a745
- `$ru-danger`: #dc3545
- `$ru-warning`: #ffc107
- `$ru-info`: #17a2b8
- `$ru-border-radius`: 4px
- `$ru-border-color`: $ru-gray-light
- `$ru-padding-sm`: 0.25rem
- `$ru-padding-md`: 0.5rem
- `$ru-padding-lg`: 1rem
- `$ru-box-shadow`: 0px 2px 4px rgba(0, 0, 0, 0.1)

## Button Base Styles

The `.ru-button` class defines the base styles for all buttons:

```scss
.ru-button {
  display: inline-block;
  font-family: $ru-font-family;
  font-size: $ru-font-size-base;
  font-weight: 600;
  padding: $ru-padding-md $ru-padding-lg;
  border: none;
  border-radius: $ru-border-radius;
  cursor: pointer;
  transition: all 0.3s ease-in-out;
  text-align: center;
  box-shadow: $ru-box-shadow;

  &:hover {
    filter: brightness(90%);
  }

  &:disabled {
    cursor: not-allowed;
    opacity: 0.6;
  }
}
```

## Button Types

The following button types are defined:

- **Primary Button**: `.ru-button--primary`
- **Secondary Button**: `.ru-button--secondary`
- **Success Button**: `.ru-button--success`
- **Danger Button**: `.ru-button--danger`
- **Warning Button**: `.ru-button--warning`
- **Info Button**: `.ru-button--info`
- **Light Button**: `.ru-button--light`
- **Dark Button**: `.ru-button--dark`

Each button type has specific background and text colors:

```scss
&--primary {
  background-color: $ru-primary;
  color: $ru-white;
}

&--secondary {
  background-color: $ru-secondary;
  color: $ru-white;
}

&--success {
  background-color: $ru-success;
  color: $ru-white;
}

&--danger {
  background-color: $ru-danger;
  color: $ru-white;
}

&--warning {
  background-color: $ru-warning;
  color: $ru-black;
}

&--info {
  background-color: $ru-info;
  color: $ru-white;
}

&--light {
  background-color: $ru-gray-light;
  color: $ru-black;
  border: 1px solid $ru-border-color;
}

&--dark {
  background-color: $ru-gray-dark;
  color: $ru-white;
}
```

## Button Sizes

The following button sizes are defined:

- **Small Button**: `.ru-button--sm`
- **Large Button**: `.ru-button--lg`

Each size has specific padding and font size:

```scss
&--sm {
  padding: $ru-padding-sm $ru-padding-md;
  font-size: 0.875rem;
}

&--lg {
  padding: $ru-padding-lg $ru-padding-lg;
  font-size: 1.25rem;
}
```

## Disabled Button

The disabled button style is defined as follows:

```scss
&--disabled {
  background-color: $ru-gray;
  color: $ru-white;
  cursor: not-allowed;
  opacity: 0.6;
}
```

## Real Usage Example

Here is an example of how to use the button styles defined in `_buttons.scss` in an HTML file:

```html [expandable]
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Button Examples</title>
    <link rel="stylesheet" href="path/to/your/compiled/css/styles.css">
</head>
<body>
    <h1>Button Examples</h1>

    <button class="ru-button ru-button--primary">Primary Button</button>
    <button class="ru-button ru-button--secondary">Secondary Button</button>
    <button class="ru-button ru-button--success">Success Button</button>
    <button class="ru-button ru-button--danger">Danger Button</button>
    <button class="ru-button ru-button--warning">Warning Button</button>
    <button class="ru-button ru-button--info">Info Button</button>
    <button class="ru-button ru-button--light">Light Button</button>
    <button class="ru-button ru-button--dark">Dark Button</button>

    <h2>Button Sizes</h2>
    <button class="ru-button ru-button--primary ru-button--sm">Small Button</button>
    <button class="ru-button ru-button--primary ru-button--lg">Large Button</button>

    <h2>Disabled Button</h2>
    <button class="ru-button ru-button--primary ru-button--disabled" disabled>Disabled Button</button>

    <h2>Outline Buttons</h2>
    <button class="ru-button ru-button--outline ru-button--primary">Outline Primary</button>
    <button class="ru-button ru-button--outline ru-button--secondary">Outline Secondary</button>
    <button class="ru-button ru-button--outline ru-button--success">Outline Success</button>
    <button class="ru-button ru-button--outline ru-button--danger">Outline Danger</button>
    <button class="ru-button ru-button--outline ru-button--warning">Outline Warning</button>
    <button class="ru-button ru-button--outline ru-button--info">Outline Info</button>
    <button class="ru-button ru-button--outline ru-button--light">Outline Light</button>
    <button class="ru-button ru-button--outline ru-button--dark">Outline Dark</button>
</body>
</html>
```

<Note>
In this example, we include various button types, sizes, a disabled button, and outline buttons to demonstrate the different styles defined in the `_buttons.scss` file. Make sure to replace `path/to/your/compiled/css/styles.css` with the actual path to your compiled CSS file.
</Note>
